# 🧠 JavaScript Architecture

The project uses modular JavaScript architecture.

## Shared Systems

| File           | Responsibility           |
| -------------- | ------------------------ |
| `main.js`      | Global systems           |
| `inventory.js` | Product data & utilities |
| `style.css`    | Shared design system     |

## Page Controllers

Each page has its own isolated logic controller:

* `shop.js`
* `product.js`
* `cart.js`
* `checkout.js`
* `reviews.js`
* `wishlist.js`

This keeps the code scalable and maintainable.

---

# 💾 Persistence Layer

AURA uses browser LocalStorage for:

* Cart data
* Wishlist
* Reviews
* Mood preferences

This creates a realistic application-like experience without a backend.

## 2. Premium Minimalism

## 3. Motion Driven UX
